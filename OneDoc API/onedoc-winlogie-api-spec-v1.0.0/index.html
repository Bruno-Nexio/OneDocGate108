    Winlogie API Spec - OneDoc sync v1.0.0

    Scroll down for code samples, example requests and responses. Select a language for code samples from the tabs above or the mobile navigation menu.

    This document specifies a few API endpoints that can be provided by the Winlogie software solution to enable a full or partial synchronization with the OneDoc online booking platform.

    Formats

    All dates and datetimes are formatted according to the ISO8601 standard:

    • A date is formatted as yyyy-MM-dd, e.g. 2018-03-19
    • A date with local time is formatted as yyyy-MM-dd[T]HH:mm:ss, e.g. 2018-03-19T10:30:00
    • A timestamp is formatted as yyyy-MM-dd[T]HH:mm:ss.SSS[Z], e.g. 2018-03-19T15:36:27.817Z where Z denotes UTC time

    Authentication

    • HTTP Authentication, scheme: basic

    Services

    The calendarId path parameter is used to target a specific Winlogie calendar, e.g. doctor X, assistant Y, room Z, etc.

    GET /calendars/{calendarId}/events

    Get calendar events

    Retrieve the events (along with patient data when available) of a given calendar within the provided date time interval.

    startDateTime >= fromDateTime AND endDateTime < untilDateTime

    This endpoint will be used to perform the initial import of events.

    Parameters

    Parameter In Type Required Description
    calendarId path string true Winlogie ID of the calendar (doctor X, assistant Y, room Z, etc.)
    fromDateTime query string true Inclusive lower bound filter for startDateTime of event (local date time in the Europe/Zurich timezone)
    untilDateTime query string true Exclusive upper bound filter for endDateTime of event (local date time in the Europe/Zurich timezone)

    DELETE /changes

    Delete the changes that were processed by OneDoc

    Delete the changes present in the folder up to the given timestamp (inclusive)

    Parameters

    Parameter In Type Required Description
    timestamp query string true Timestamp up to which changes should be deleted inclusively (change timestamp <= timestamp parameter)

    Example responses
